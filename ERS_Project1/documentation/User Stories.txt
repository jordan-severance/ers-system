Expense Reimbursement System (ERS)

A-An Employee can login
C-An Employee can view the Employee Homepage
E-An Employee can logout
M-An Employee can submit a reimbursement request
N-An Employee can view their pending reimbursement requests
P-An Employee can view their resolved reimbursement requests
Q-An Employee can view their information
O-An Employee can update their information
-An Employee receives an email when one of their reimbursement requests is resolved (optional)

B-A Manager can login
D-A Manager can view the Manager Homepage
F-A Manager can logout
H-A Manager can approve/deny pending reimbursement requests
I-A Manager can view all pending requests from all employees
J-A Manager can view all resolved requests from all employees and see which manager resolved it
K-A Manager can view all Employees
L-A Manager can view reimbursement requests from a single Employee
G-A Manager can register an Employee, which sends the Employee an email with their username and temp password (optional)
     -An Employee can reset their password (Optional - tied with above functional requirement)
